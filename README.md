A design review used to mean exporting the model, waiting on a render farm, and presenting a static image that nobody could walk through. Enscape collapses that to one button: press play in Revit and the same geometry is immediately explorable in real time on screen or in a headset.

### Supported host applications

| Application | Native plugin |
|---|---|
| Autodesk Revit | 2021–2026 |
| SketchUp | 2021–2026 |
| Rhinoceros | 6, 7, 8 |
| ArchiCAD | 25–28 |
| Vectorworks | 2022–2026 |

### Live sync model

```
Any change in the host (moved wall · new material · adjusted light)
    ↓
Enscape viewport updates in < 0.5 seconds
    ↓
No export · No file transfer · No re-import
```

### Output modes

- **Real-time viewport** — navigate at 60fps during design sessions
- **Still render** — path-traced quality output (up to 8K)
- **Video fly-through** — GPU-accelerated animation with depth of field
- **360° panorama** — equirectangular for client web viewer
- **VR walkthrough** — live or pre-recorded tour in headset
- **Standalone EXE** — shareable executable walkthrough, no Enscape licence required

### Asset library

10,000+ photorealistic objects: furniture · people · trees · vehicles · lights — drag directly from the Enscape library panel into the host
